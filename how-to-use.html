Welcome to PlotMe wiki.

  1. Put the MyPlot.phar in your plugins folder
  2. Run your server to generate the MyPlot folder
  3. Change settings in the config.yml file in the MyPlot folder
  4. Use the command, /plotme reload, to reload the settings.

The world names in the config.yml files must be in lowercase! That is extremely important.
You must have your worlds defined in both the Generator config and the PlotMe config.
